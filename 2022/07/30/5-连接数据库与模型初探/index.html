连接数据库与模型初探

[^]:

一、连接数据库

  1. ThinkPHP 采用内置抽象层将不同的数据库进行封装处理

  2. 数据抽象层基于 PDO 模式，无需针对不同的数据库编写相应的代码

    • PDO

    全称为（PHP data object）代表PHP和数据库服务之间的一个连接，可以视作针对不同问题的解决方案，是一个标准的

    API（应用程序编程接口）。

  3. 在根目录的 config 下的 database.php 可以设置数据库连接信息

  4. 本地测试，会优先采用 .env 的配置信息，与 database 配置对应即可

  5. 可以通过删除改变 .env 的配置，或删除 .env 来验证 database 的执行优先级

  6. database .php 配置中， dafault 表示设置默认的数据库连接

  7. connections 配置数据库连接信息，可以是多个数据库，便于切换

二、模型初探

  1. User 继承模型基类，即可实现数据调用

  2. 受保护的字段$connection，则是切换到 demo 数据库
