Laravel的安装

[^* 前提：安装了PHP study套件 *]:

一、安装composer

下载地址：composer安装🛒

二、配置镜像

命令提示符输入：

➡**composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/**

三、安装Laravel项目

命令提示符输入：

➡**composer create-project --prefer-dist laravel/Laravel=6 D:\Laravel**

四、启动Laravel框架

  1. 打开PHP study面板，网站一栏点击创建网站

  2. 编辑基本配置，域名格式为:”##.##”,根目录选择刚刚安装好的Laravel项目下的public文件夹。 eg：baidu.com

  1. 编辑路径下”nginx.htaccess“文件

    输入以下代码：

    location / {
    # First attempt to serve request as file, then
    # as directory, then fall back to displaying a 404.
    # try_files $uri $uri/ =404;
    ​ try_files $uri $uri/ /index.php?$query_string;
    }

  2. 浏览器访问设置好的域名，就可以进入Laravel项目的原始页面了。
