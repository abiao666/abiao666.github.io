Node.js 安装

Created2022-07-20|Updated2022-07-20
|Post View:

Node.js

[^]:

1、Node.js 环境的安装

安装包可前往官网首页Node.js直接下载

2、区分 LTS 版本和 Current 版本的不同
  1. LTS 为长期稳定版，对于追求稳定性的企业及项目，推荐安装 LTS 版本的 Node.js。
  2. Current 为新特性尝鲜版 ，对热衷于尝试新特性的用户来说，推荐安装 Current 版本的 Node.js。但是可能存在 bug 或安全漏洞。
3、查看已安装的 Node.js 的版本号

win + r 打开运行面板，输入cmd打开终端，输入 node -v 回车，即可看到已安装的 Node.js 的版本号

  • node -v之间有一个空格。
