控制器定义

[^]:

一、控制器定义

控制器

controller，控制器文件存放再 controller 目录下

  1. 改变系统默认的控制器文件目录，可以在 config 下 route.php

    配置：controller_layer => 更改的目录

  2. 类名和文件名大小写保持一致，采用驼峰式（首字母大写）

  3. ，避免引入同类名冲突，route.php 设置控制器后缀

二、渲染输出

  1. 直接采用方法内 return 返回的方式直接就输出了

  2. 使用 json 输出，直接采用 json函数

    $data = array[‘a’ => 1, ‘b’ => 2, ‘c’ => 3];

    return json($data);

  3. 不推荐使用 die、exit 等 PHP 方法中断代码执行，推荐助手函数halt();

    halt(‘中断测试’);

三、基础控制器

仅仅提供了控制器验证功能，并注入了 think\App 和 think\Request;

  • 继承

四、空控制器

单应用模式下，我们可以给项目定义一个 Error 控制器类，来提醒错误；

五、多级控制器

多级控制器就是再控制器 Controller 目录下在建立目录并创建控制器
